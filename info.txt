Описание программы для заказов обедов в офис

Для работы программы нужны 3 файла с данными:
    source_xml/menu.xml - список блюд из меню
    source_xml/clients.xml - список заказчиков
    source_xml/orders.xml - список заказов (1 заказ = 1 блюдо для одного клиента, в файле указаны ClientId (id заказчика из файла clients.xml) и DishId (id блюда из файла menu.xml))
